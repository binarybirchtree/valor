# Vindicator

An AI for [Vindinium](http://vindinium.org) that explores the use of HTTP APIs with C++.

## Building

*  Install dependencies.

   ```Shell
   sudo apt-get install clang make libcpprest-dev libgoogle-glog-dev libgtest-dev
   ```

*  Build.

   ```Shell
   make
   ```

*  Run tests.

   ```Shell
   make test
   ```
